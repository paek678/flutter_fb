# 게임 공략 앱 DAKGG(던전앤파이터)


<br>

## 게임 공략 앱
던전앤파이터 게임과 관련된 다양한 정보를 모바일로 쉽고 빠르게 확인할 수 있는 앱



<br>

## 팀원 및 역할
팀장
<br>
20 최재서 (백엔드)
<br>

팀원
<br>
20 백성열 (백엔드)
<br>
20 김형규 (프론트)
<br>
20 안수현 (프론트)



<br>

## 사용 방법
기본적으로 직관적인 앱이라 기능별로 쉽게 사용가능
<br>

캐릭터 검색: 게임 내 캐릭터 이름 검색 => 해당 캐릭터 관련 정보 확인 가능( 예시 : 검색창에 츄르민초 검색 )
<br>

순위: 서버별, 직업 별 순위 확인 가능
<br>

경매장: 아이템 이름 검색 시 아이템 정보, 시세 및 그래프 추이 확인 가능( 예시 : 아이템 검색창에 마법으로 봉인된 검색 )
<br>

커뮤니티 및 공지사항: 유저들끼리의 소통(게시글 작성과 댓글), 각종 소식들 직접 확인 가능 ( 게시글 클릭시 세부 내용 확인 및 댓글 작성 가능 )



<br>

## 주의사항
현재 일부 가상 모바일 에뮬레이터에서는 구글 로그인 기능이 작동하지 않음
<br>
=> 구글 버전이 맞지 않아서 로그인 토큰까지는 생기나 로그인 접속이 불가능

<br>

해결방안
<br>

1. 가능한 일부 가상 모바일 에뮬레이터를 사용해도 됨
<br>

2. 가급적이면, 가상 모바일 에뮬레이터 보다는 실제 기기를 사용하는 것을 권장
